CSV Format Requirements

Your CSV file should include the following columns:

  • firstName - Required. Member's first name
  • lastName - Optional. Member's last name
  • officialNumber - Optional. Unique member number (auto-generated if not provided)
  • address - Optional. Member's address
  • plateNumber - Optional. Vehicle plate number
  • description - Optional. Additional notes

Only firstName is required. Official numbers will be automatically assigned if left empty.
